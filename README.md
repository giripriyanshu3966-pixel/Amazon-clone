# Amazon Clone

A front-end clone of the Amazon homepage built using HTML5 and CSS3. This project was created to practice web development fundamentals, including layout design, responsive styling, Flexbox, and UI development.

## Features

- Responsive Navigation Bar
- Search Section
- Hero Banner
- Product Category Cards
- Shopping Panels
- Footer Section
- Clean and Structured UI
- Responsive Design Principles

## Technologies Used

- HTML5
- CSS3

## Learning Outcomes

Through this project, I learned:

- Semantic HTML Structure
- CSS Flexbox
- Responsive Web Design
- Layout and Positioning Techniques
- UI Development Best Practices
- Code Organization
